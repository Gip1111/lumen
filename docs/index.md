# Welcome to Lumen Documentation

Lumen is an Arch-based Linux distribution with a deeply integrated AI assistant.

## Core Features
- **Lumen AI Assistant**: A natural-language interface for system management.
- **Hardware First**: Automatic driver detection and easy installation.
- **Windows Approachability**: Designed for users coming from other operating systems.

## Getting Started
1. [Download the ISO](https://github.com/Gip1111/lumen/releases)
2. Create a bootable USB
3. Boot and use the **Calamares** installer.

## Using the AI Assistant
Press `Super + Space` to summon the assistant. You can ask things like:
- "What hardware am I running?"
- "Check for updates"
- "Help me install drivers for my graphics card"
